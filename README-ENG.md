# **BarranquillaMotors – Automotive Parts Website (HTML + Bootstrap 5)**

A **modern, elegant, and 100% responsive** website for an auto parts store in Barranquilla, built with **pure HTML and Bootstrap 5**!

### **What this page does**

It is a **complete landing-page–style website** with everything an auto parts store needs:

* **Fixed navbar** with responsive menu (turns into a hamburger on mobile)
* **Main carousel** with 3 promotional slides (engines, parts, electric vehicles)
* **Services section** with 3 stylish cards
* **Benefits icons**: free shipping, refunds, gifts
* **Product catalog** with 8 cards (rims, steering wheels, engines, locks, etc.)
* **Promotional banner** with image
* **Functional and stylish contact form**
* **Full footer** with:

  * About Us
  * Useful links
  * Contact + social media
  * Embedded Google Map
  * Reserved rights

### **Technologies used**

* Semantic HTML5
* **Bootstrap 5.3** (CDN) → professional design without writing CSS
* **Bootstrap Icons** → modern icons
* Embedded Google Maps
* Fully **responsive** (looks perfect on mobile, tablet, and desktop)

### **Key features**

* Elegant dark design (very modern)
* Carousel with autoplay and controls
* Cards with shadows and hover effects
* Centered, well-styled form
* Social media with real icons
* Clean and well-commented code (great for learning!)
