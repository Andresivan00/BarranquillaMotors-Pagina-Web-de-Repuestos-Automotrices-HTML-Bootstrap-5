# BarranquillaMotors – Página Web de Repuestos Automotrices (HTML + Bootstrap 5)

¡Una página web **moderna, elegante y 100% responsive** para una tienda de repuestos de autos en Barranquilla, hecha con **HTML puro y Bootstrap 5**!

### Qué hace esta página

Es una **página completa tipo landing page** con todo lo que necesita una tienda de repuestos:

- **Navbar fija** con menú responsive (se convierte en hamburguesa en móviles)
- **Carrusel principal** con 3 slides promocionales (motores, repuestos, vehículos eléctricos)
- **Sección de servicios** con 3 tarjetas bonitas
- **Íconos de beneficios**: envío gratis, reembolso, regalos
- **Catálogo de productos** con 8 tarjetas (rines, volantes, motores, cerraduras, etc.)
- **Banner promocional** con imagen
- **Formulario de contacto** funcional y bonito
- **Footer completo** con:
  - Acerca de nosotros
  - Links útiles
  - Contacto + redes sociales
  - Mapa de Google embebido
  - Derechos reservados

### Tecnologías usadas
- HTML5 semántico
- **Bootstrap 5.3** (CDN) → diseño profesional sin escribir CSS
- **Bootstrap Icons** → íconos modernos
- Google Maps embebido
- Totalmente **responsive** (se ve perfecto en celular, tablet y PC)

### Características destacadas
- Diseño oscuro elegante (muy moderno)
- Carrusel con autoplay y controles
- Tarjetas con sombras y hover
- Formulario centrado y con estilo
- Redes sociales con íconos reales
- Código limpio y bien comentado (¡perfecto para aprender!)

### Ideal para
- Proyectos finales de desarrollo web
- Portafolio profesional
- Tiendas reales de repuestos
- Aprender Bootstrap 5 desde cero
- Clases de HTML/CSS

**100% estática** – solo abre el archivo `index.html` y funciona al instante  
No necesitas servidor ni base de datos
